# Gayathri Vasamsetti — Portfolio

Personal portfolio site built with vanilla HTML, CSS, and JavaScript, featuring an interactive 3D visualization of my AI/dev tech stack built with Three.js.

## 🔗 Live Site
[https://gayathri-ai.netlify.app/](https://gayathrivasamsetti555-lab.github.io/MyPortfolio/)

## 📁 Project Structure

```
MyPortfolio/
├── index.html              # Main page markup
├── css/
│   └── style.css           # All styling (CSS custom properties, layout, responsive rules)
├── js/
│   └── main.js             # Three.js 3D hero visualization
├── assets/
│   └── resume/
│       └── Gayathri_Vasamsetti_Resume.pdf   # Downloadable resume
└── README.md
```

## 🛠️ Built With
- HTML5 / CSS3 (custom properties, CSS Grid, Flexbox)
- Vanilla JavaScript
- [Three.js](https://threejs.org/) — 3D hero visualization
- Google Fonts (Space Grotesk, Inter, JetBrains Mono)

## ✨ Features
- Interactive 3D node graph in the hero, built from my actual tech stack (Claude, Gemini, RAG, LangChain, React, .NET) as an embedding-space visualization
- Fully responsive (mobile → desktop)
- Respects `prefers-reduced-motion`
- Downloadable resume linked directly in nav and hero

## 🚀 Running Locally

No build step required. Either:

```bash
# Just open it
open index.html

# Or serve it (recommended, closer to production)
python3 -m http.server 8000
# then visit http://localhost:8000
```
